# webui-forge-maestro

A personal Python MCP server for driving Stable Diffusion WebUI Forge via its
`/sdapi/v1/*` HTTP API.

> **Status: experimental.** A from-scratch personal port of
> [`Ichigo3766/image-gen-mcp`][upstream] — built primarily to learn the MCP
> protocol surface and as a foundation for personal tweaks. **Not** trying to
> replace the upstream; if you want a battle-tested image-gen MCP, use that
> one.
>
> Full README arrives once v1 is feature-complete.

[upstream]: https://github.com/Ichigo3766/image-gen-mcp
